# Pharm System - Backend

This folder contains a minimal scaffold for the backend API.

Structure:

- `src/app.js` - Express app
- `src/server.js` - Server entry
- `src/config` - env and supabase client
- `src/middlewares` - auth/role/error middlewares
- `src/routes` - route definitions
- `src/controllers` - controllers
- `src/services` - services/business logic
- `src/utils` - helpers
- `src/validations` - validation helpers

Quick start:

```bash
cd backend
npm install
npm start
```
